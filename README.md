# Coders-Garage
A react coding website made for the Summer of Innovation Web Development project.  
[Problem statement Link](https://drive.google.com/file/d/1N5pfnjgjo8eoB571Ia0KvBgDLbKP0SPG/view?usp=sharing)

# To run the client
```
  cd client
  npm i
  npm start
```

# To run the server
```
  cd server
  npm i
  npm run dev
```
